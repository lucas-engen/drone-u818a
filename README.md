- Finalize a instalação normalmente.
2) Clique no botão de copiar para o clipboard:  
[Imgur](https://i.imgur.com/RQTgnad.jpg)  
3) Vá para a área de trabalho (ou outra pasta de sua escolha), clique com o botão direito em um espaço vazio e depois clique em **Git Bash Here**:  
[Imgur](https://i.imgur.com/HkD2j8d.jpg)    
4) Uma tela preta abrirá. Após isso, digite o seguinte comando:  
**git clone https://github.com/lucas-engen/drone-u818a.git**  
[Imgur](https://i.imgur.com/GsShRHO.jpg)  
Aperte enter e aguarde até que ele seja executado. A saída no console será similar a isso:  
[Imgur](https://i.imgur.com/7Ra9rE7.jpg)  
5) Pronto! Uma pasta com o nome do projeto será criada. Nela estarão todos os arquivos:  
[Imgur](https://i.imgur.com/fGFPBbD.jpg)  

## Como enviar as suas alterações
1) Vá ao site aonde se encontra o projeto, e clique no botão **upload files**:  
[Imgur](https://i.imgur.com/lJdwfos.jpg)  
2) Clique no botão indicado pela seta azul, escolha os arquivos no seu computador que foram alterados. Após isso, clique no botão indicado pela seta verde. Se quiser, coloque uma mensagem qualquer sobre quais alterações você realizou:  
[Imgur](https://i.imgur.com/w8xW0qO.jpg)
